# AGENTS.md

## 项目概述

本仓库不是软件工程项目，而是一份按主题整理的**中医古籍参考资料库**（EPUB 格式）。
其目的是为 Agent（及人类读者）在回答中医相关问题时提供可溯源的原始依据。

Agent 在处理中医问题时，应遵循"**道—法—方—药—术**"的知识分层（参见
`TCM.md`），并优先从本仓库各分类目录下的典籍中查找规律与原理，再据此组织答案。
凡涉及原理性、规律性的论断，都应尽量标明出处（书名、篇目）。

由于没有源代码，AGENTS.md 中常见的"构建命令 / 代码风格 / 测试 / 安全加固 /
配置管理"等章节不适用，本文档聚焦于资料分类、作用与使用规则。

## 目录分类、作用与查阅指引

所有文件均为 `.epub`。目录名为中文，表示所含典籍的主题域。

### `古文翻译/` — 古文字词释义
- **作用**：阅读其他四类典籍遇到生僻字、词义存疑时的训诂工具。
- **使用时机**：当原文出现一字多义、通假、古今异义，或 Agent 对某个字/词
  的训释不确定时，先到本目录查字再作解释。
- **资料**：
  - `古汉语常用字字典(第4版).epub` — 古汉语常用字字典，第 4 版。

### `病理/` — 人体运行规律（道）
- **作用**：解释人体如何运行、为何生病、生病后如何反应，是中医的"原理书"。
  是一切辨证与处方背后的理论根基。
- **核心规律（出处：《黄帝内经》）**：
  - **阴阳五行**：万物与人体脏腑相生相克的动态平衡。
  - **藏象经络**：五脏六腑作为"功能系统"而非解剖器官；经络为气血运行通道。
  - **天人相应**：人体随四季、昼夜、地域变化调整。
  - **治未病**：强调预防优于治疗。
- **使用时机**：回答"为什么会生病 / 为什么会有某种反应 / 某脏腑的功能是什么"
  等原理性问题时，引用本目录。
- **资料**：
  - `黄帝内经（全二册）.epub` — 《黄帝内经》（素问 + 灵枢），文白对照，姚春鹏
    译注。

### `药理/` — 药物的自然属性规律（药）
- **作用**：解释每一味药的性味、归经、功效及配伍规律，相当于中药学的"元素
  周期表"。此目录同时收录《难经》译注，可用于脏腑经络疑难的答疑。
- **核心规律（出处：《神农本草经》）**：
  - **三品分类**：上品养命、中品养性、下品治病（共 365 味药）。
  - **四气五味**：寒、热、温、凉 + 酸、苦、甘、辛、咸，决定药物作用方向。
  - **七情配伍**：单行、相须、相使、相畏、相恶、相反、相杀。
- **使用时机**：回答"某味药药性如何 / 能否与另一味同用 / 归哪条经"等问题
  时，引用本目录；《难经》用于补充《内经》中关于脏腑、经脉、脉法的疑难条文。
- **资料**：
  - `神农本草经译注·难经译注.epub` — 张树生、包来发译注，中医古籍名著编译
    丛书。

### `处方思路/` — 辨证施治规律（法 / 方）
- **作用**：把《内经》的理论、 《本草经》的药性落地成"可操作的方剂"；教 Agent
  如何按证型组方。
- **核心规律**：
  - **六经辨证**（出处：《伤寒论》）：太阳、阳明、少阳、太阴、少阴、厥阴——
    病邪由表入里的六个阶段，每一证对应明确方剂，体现"理—法—方—药"一体化。
  - **脏腑辨证与三因论雏形**（出处：《金匮要略》）：以脏腑功能失调为纲；病因
    分为外因、内因、不内外因；大量经典方的源头。
- **使用时机**：回答"这个证该开什么方 / 某方治什么 / 为何如此配伍"时，引用
  本目录；外感病优先查《伤寒论》，内伤杂病优先查《金匮要略》。
- **资料**：
  - `伤寒杂病论（简体中文版）.epub` — 张仲景《伤寒杂病论》简体中文版。
  - `金匮要略.epub` — 张仲景《金匮要略》。

### `穴位经络/` — 经络与针灸操作规律（术）
- **作用**：解释经络腧穴体系与针灸操作原理，是针灸临床的集大成之作。
- **核心规律（出处：《针灸大成》）**：
  - **经络腧穴系统**：十二正经 + 奇经八脉 + 361 个经穴的定位、主治。
  - **刺法灸法**：补泻手法（烧山火、透天凉等）、进针深浅、留针时间。
  - **配穴处方**：如"四总穴歌""回阳九针歌"等口诀化的临床经验。
  - **医案**：杨继洲本人大量临床案例，是理论落地的示范。
- **使用时机**：回答"某穴位置 / 主治 / 针刺补泻 / 配穴方案"等问题时，引用本
  目录；对原著晦涩处可参照倪海厦同步文稿作为现代讲解补充。
- **资料**：
  - `针灸大成.epub` — 明·杨继洲编撰，夏魁周校注。
  - `倪海厦《针灸大成》视频同步文稿.epub` — 倪海厦讲《针灸大成》视频的同步
    文稿，作为现代讲解与教学辅助。

## 强制前置规则（MUST）

在回答任何中医相关问题前，Agent 必须按顺序完成以下动作，任何一步都不得跳过：

1. **必须先查典籍**：根据问题层级定位目录（见下节工作流），实际打开对应
   `.epub` 检索相关条目。未经检索不得作答。
2. **epub 读取失败时必须明示**：若因缺少 EPUB 解析器等原因无法读取原文，
   必须在回答中**显式声明"未查证原文"**；你可以通过网络搜索来补充典籍未提及到的内容，但 **需要向用户说明**。
3. **禁止伪托出处**：未实际查阅到的条文、篇目、原文引用一律不得出现在
   回答中。笼统写"参见《XX》"而未核对原文，视同伪托，属于"臆造"。
4. **食疗 / 养生 / 日常保健类问题同样适用**：不因问题"偏生活化"而降低
   查证标准。凡涉及性味、归经、功效、配伍的论断，均须回到 `药理/` 等
   目录核对。

## Agent 回答问题的工作流

遇到中医相关问题时，按以下步骤组织答案：

1. **判定问题层级**，决定查阅目录：
   - 原理 / 为什么 → `病理/`（《黄帝内经》）
   - 药性 / 配伍 → `药理/`（《神农本草经》；脏腑脉法疑难兼查《难经》）
   - 辨证 / 开方 → `处方思路/`（外感《伤寒论》、内伤《金匮要略》）
   - 经络 / 针灸 → `穴位经络/`（《针灸大成》+ 倪海厦文稿）
   - 古文字词 → `古文翻译/`（《古汉语常用字字典》）
2. **遵循规律**：依据该目录核心规律（见上文各节）推演答案，不要脱离这些经典
   体系凭空发挥。
3. **标注出处**：涉及原理、规律、条文、方剂、穴位主治时，引用书名；若能指出
   具体篇章（如《素问·阴阳应象大论》《伤寒论·太阳病篇》）则更佳。
4. **跨目录协同**：复杂问题常需多目录协同。例如一个外感处方问题：
   - 在 `病理/` 查阴阳表里寒热的判断依据；
   - 在 `处方思路/` 查六经辨证归类与对应方；
   - 在 `药理/` 核对每味药的性味归经；
   - 必要时在 `古文翻译/` 核对条文字词。
5. **拒绝臆造**：若所查目录未涵盖该问题，明确告知用户来源不足，而不是
   编造；**未经核对而笼统引用书名同样属于臆造**。

## 推荐的阅读 / 查证顺序（零基础）

参照 `TCM.md`：**原理 → 药物 → 临床 → 操作**

1. `病理/`《黄帝内经·素问》选读 — 建立阴阳五行、藏象经络的思维框架。
2. `药理/`《神农本草经》选读 — 理解药性语言。
3. `处方思路/`《伤寒论》 — 学会用六经给外感病归类。
4. `处方思路/`《金匮要略》 — 扩展到内伤杂病的脏腑辨证。
5. `穴位经络/`《针灸大成》 — 掌握经络腧穴与针刺操作。
6. `古文翻译/`《古汉语常用字字典》 — 在上述过程中随查随用。

## 文件处理约定

- 所有资料为 `.epub`，如需程序化读取需自行准备 EPUB 解析器（如 Python
  `ebooklib`），可尝试使用`.venv`下的工具或其他方式来读取 EPUB 内容。
- 文件名包含中文与括号，在 Shell 中传参时务必加引号。
- 资料只读：不得修改、重排或对外再分发 EPUB 内容。
